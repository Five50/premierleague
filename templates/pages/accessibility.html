2. {% trans "Accessibility Standards" %}

{% trans "Our website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA. These guidelines explain how to make web content more accessible to people with disabilities." %}

{% trans "WCAG 2.1 Principles" %}

{% trans "Perceivable" %}

{% trans "Information must be presentable in ways users can perceive" %}

{% trans "Operable" %}

{% trans "Interface components must be operable by all users" %}

{% trans "Understandable" %}

{% trans "Information and UI operation must be understandable" %}

{% trans "Robust" %}

{% trans "Content must be robust enough for various assistive technologies" %}
